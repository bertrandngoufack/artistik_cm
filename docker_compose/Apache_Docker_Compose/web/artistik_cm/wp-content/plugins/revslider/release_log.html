Version 7.0.9 (17th April 2026)

New Features
  • Layers hidden in the editor (because they are not part of the active scene) can now be edited by temporarily resetting parent transitions and visibility during editing.
  • In the editor, full-stage alignment is now calculated based on the visible checkerboard stage area (instead of the full wrapper container) to make elements easier to edit.
  • Added an optional live preview rendering feature in the Elementor Builder
  • Added a function to add bulk slides from WP media library
  • Added a function to add Slide(s) from existing modules
Changes
  • Renamed Element to Layer for consistent terminology
  • Removed none working WPML class and replaced it with a new WPML functionality through the AI Translate AddOn
Bug Fixes
  • Fixed carousel touch swipe not working on Android devices.
  • SR7.F.module.collect() now reinitializes fast Ajax-loaded content after page changes.
  • Fixed the default action link target so links open in the same window by default.
  • Action setup is now guided: the trigger selector stays disabled until an action type is selected.
  • The slide background can now be used as an action trigger.
  • Fixed multi-selected layers not moving with the keyboard arrow keys.
  • Improved stability on fast Ajax/PJAX sites where SR7 could fail to start after navigation.
  • Fixed modules failing to load in the Divi Builder after upgrading.
  • Fixed a typo in the video settings.
  • Removed the full-screen button from video controls when Allow Full Screen is disabled.
  • Fixed Uniform Gaps not appearing in preview and preventing unexpected spacing issues.
  • Fixed a bug where video thumbnails were regenerated whenever the source changed, even if a matching thumbnail already existed in the WordPress Media Library.
  • YouTube, HTML5, and Vimeo videos now support contain/cover sizing based on the selected aspect ratio while keeping the original layer dimensions.
  • Fixed a backward compatibility issue caused by themes/plugins calling the removed getAllSliderAliases() function.
  • Fixed scroll-based filters and transforms rendering even when scroll effects were disabled globally.
  • Fixed an issue where a missing canvas could break nested modals and render content with unwanted empty space.
  • Fixed duplicated layers in the editor sometimes receiving multiple chained _DBL_ random suffixes.
  • Fixed an issue where Disabled on Mobile could disable the entire module, even when the module was not set to run on mobile.
  • Fixed missing Alt and Title attributes in noscript image elements generated by Slide Background layers.
  • Improved Slide ID handling: duplicate Slide IDs are now updated in the editor and frontend to prevent broken processing.
  • Scroll Based Timeline interactions were not started in some situations
